## Instalação

Instruções para inicializar e configurar o projeto.

```sh
# Navegue até o diretório do projeto
cd frontend

# Instale as dependências
npm install
npm run dev

# Navegue até o diretório do projeto
cd backend

# Instale as dependências
npm install
npm start
```
